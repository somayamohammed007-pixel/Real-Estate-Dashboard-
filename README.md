# Real Estate Dashboard

## Overview
An interactive Power BI dashboard designed to analyze real estate market trends, property pricing, ownership types, amenities, and location insights.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key KPIs
- Total Properties
- Average Property Price
- Average Area
- Average Price per Sqft
- Furnished Property Count

## Dashboard Pages

### 🏠 Market Overview
- Property Distribution by Type
- Ownership Analysis
- Property Count Analysis

### 💰 Price Analysis
- Average Price by City
- Average Price by Bedrooms
- Average Price by Furnishing Type
- Price vs Covered Area

### ⭐ Amenities Analysis
- Most Common Amenities
- Smart Home Analysis
- Property Features Insights

## Key Insights
- Property prices increase with larger covered areas.
- Premium cities show significantly higher average property prices.
- Furnished properties tend to have higher market values.
- Smart-home enabled properties command higher prices.
- Amenities contribute positively to property valuation.
